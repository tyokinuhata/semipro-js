Semipro JS

この書籍は, 中級者向けのJavaScriptの解説書である.
すなわち, 読者は入門書を読み終え, いくつかJavaScriptでの開発を経験しているものとして説明している.

このリポジトリを編集する場合は, 開発者ガイド を参照してください. このリポジトリに関する質問や本書の改善に関する提案等は Issues までよろしくお願いします.
